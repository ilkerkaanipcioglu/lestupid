# Roadmap

## Faz 0: Iskelet

- Monorepo klasorleri olustur.
- Expo mobil app iskeleti kur.
- Phoenix API iskeleti kur.
- Health endpoint ekle.
- Mock Kadikoy/Nairobi gorevlerini mobilde goster.
- LesTupid manifest, runtime mode ve portability bilgilerini dogrula.

## Faz 1: Ilk Oynanabilir MVP

- Mobil harita kutuphanesini ekle.
- Gorev pinlerini haritada goster.
- Gorev detay ve tamamlama akisini bagla.
- Backend'de cities, places, quests ve completions kaynaklarini gercek API olarak ac.
- Basit puan ve rozet kurallarini uygula.
- Aktivasyon/consent yuzeyi ekle: Les Poke, location, optional Les Match,
  optional value proof.

## Faz 2: Icerik ve Esnaf

- Seed verileri genislet.
- Esnaf odulu alanlarini ekle.
- Sponsor gorevlerini ayirt et.
- Basit admin icerik girisi ekle.

## Faz 3: Sosyal ve Canli Katman

- Kullanici uretimli hikaye/oneri akislarini ekle.
- Moderasyon kuyrugu tasarla.
- Anlik etkinlik pinleri icin Urban Live prototipi ekle.

## Faz 4: Ekosistem Adapterlari

- Les Match icin check-in, quest interest, event plan ve travel plan sinyalleri.
- Les AI/AgentAndBot icin quest idea, field note ve place evidence adapterlari.
- Les Block icin opsiyonel quest/badge/point proof eventleri.
