# LesTupid

LesTupid, Harezm ekosisteminin sadakat, sertifikasyon ve bekleme/queue
katmanlarini tasiyan cati alandir.

## Guncel Yapi

```text
LesTupid/
  certification/
  waiting/
```

| Klasor | Rol |
| --- | --- |
| `certification/` | Lestupid felsefesi, sertifikasyon kriterleri, loyalty network ve urun sertifika registry dosyalari. |
| `waiting/` | Beklemesiz / akilli sira ve bekleme uygulamasi, openspec belgeleri ve prototip dosyalari. |
